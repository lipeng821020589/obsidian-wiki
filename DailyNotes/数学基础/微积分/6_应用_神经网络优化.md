# 神经网络优化中的微积分

## 神经网络的优化问题

### 定义
$$\min_{\mathbf{W}} \mathcal{L}(\mathbf{W}) = \frac{1}{N}\sum_{i=1}^N \ell(f(\mathbf{x}_i; \mathbf{W}), \mathbf{y}_i)$$

其中：
- $\mathbf{W}$：所有可学习参数
- $\ell$：损失函数
- $f$：神经网络
- $(\mathbf{x}_i, \mathbf{y}_i)$：训练样本

---

## 梯度计算

### 单层梯度

对于全连接层 $\mathbf{y} = \sigma(\mathbf{x}\mathbf{W} + \mathbf{b})$：

```python
# 前向
z = x @ W + b
y = sigmoid(z)

# 反向
dL/dz = dL/dy * y * (1 - y)  # sigmoid导数
dL/dW = x.T @ dz
dL/db = dz.sum(dim=0)
```

### 深层网络
链式法则逐层反向传播。

---

## 激活函数的导数

### Sigmoid
$$\sigma(x) = \frac{1}{1 + e^{-x}}, \quad \sigma'(x) = \sigma(x)(1 - \sigma(x))$$

**特点**：输出在(0,1)，梯度在x=0最大。

### Tanh
$$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}, \quad \tanh'(x) = 1 - \tanh^2(x)$$

**特点**：输出在(-1,1)，零中心。

### ReLU
$$\text{ReLU}(x) = \max(0, x), \quad \text{ReLU}'(x) = \begin{cases} 1 & x > 0 \\ 0 & x \leq 0 \end{cases}$$

**特点**：计算高效，但x<0时梯度为0（神经元死亡）。

### Leaky ReLU
$$\text{LeakyReLU}(x) = \begin{cases} x & x > 0 \\ \alpha x & x \leq 0 \end{cases}, \quad \alpha \approx 0.01$$

### GELU
$$\text{GELU}(x) = x \cdot \Phi(x), \quad \Phi \text{ 是标准正态CDF}$$

BERT/GPT使用，数值稳定实现：
```python
# ▶ GELU 激活函数
import torch
import torch.nn.functional as F

x = torch.randn(3)

# PyTorch 内置
print(F.gelu(x))  # tensor([...])

# 手动实现
def gelu_manual(x):
    return 0.5 * x * (1 + torch.tanh(torch.sqrt(torch.tensor(2/3.14159)) * (x + 0.044715 * x**3)))

print(gelu_manual(x))
```

## 损失函数的梯度

### MSE (Mean Squared Error)
$$\mathcal{L}_{\text{MSE}} = \frac{1}{N}\sum_i (y_i - \hat{y}_i)^2$$

**梯度**：
$$\frac{\partial \mathcal{L}}{\partial \hat{y}_i} = \frac{2}{N}(y_i - \hat{y}_i)$$

### Cross-Entropy + Softmax（最常用）
```python
# ▶ CrossEntropy + Softmax 梯度
import torch
import torch.nn.functional as F

logits = torch.randn(4, 10, requires_grad=True)
target = torch.tensor([2, 5, 0, 7])

loss = F.cross_entropy(logits, target)
loss.backward()

print(f"Loss: {loss.item():.4f}")
print(f"Grad shape: {logits.grad.shape}")  # (4, 10)
```

**优势**：梯度稳定，不会饱和。

### BCE (Binary Cross-Entropy)
$$\mathcal{L}_{\text{BCE}} = -\frac{1}{N}\sum_i [y_i \log \hat{y}_i + (1-y_i)\log(1-\hat{y}_i)]$$

**梯度**：
$$\frac{\partial \mathcal{L}}{\partial \hat{y}_i} = \frac{\hat{y}_i - y_i}{\hat{y}_i(1-\hat{y}_i)}$$

**注意**：当 $\hat{y}_i \to 0$ 或 $1$ 时梯度爆炸。

## 梯度消失与梯度爆炸

### 梯度消失 (Vanishing Gradient)
**原因**：多层小梯度累积。

Sigmoid的饱和区（x<-4或x>4）梯度接近0：
$$\sigma'(x) < 0.01 \text{ when } |x| > 4$$

**解决方案**：
- ReLU/LeakyReLU
- BatchNorm（稳定梯度）
- 残差连接（ResNet）
- LSTM/GRU的门控机制

### 梯度爆炸 (Exploding Gradient)
**原因**：多层大梯度累积。

**解决方案**：
- 梯度裁剪（`clip_grad_norm_`）
- 权重初始化（Xavier/Kaiming）
- BatchNorm

## 权重初始化

### Xavier初始化
适用于Sigmoid/Tanh：
$$\mathbf{W} \sim U\left(-\sqrt{\frac{6}{n_{\text{in}} + n_{\text{out}}}}, \sqrt{\frac{6}{n_{\text{in}} + n_{\text{out}}}}\right)$$

或高斯版本：
$$\mathbf{W} \sim \mathcal{N}\left(0, \frac{2}{n_{\text{in}} + n_{\text{out}}}\right)$$

### Kaiming初始化（He初始化）
适用于ReLU：
$$\mathbf{W} \sim \mathcal{N}\left(0, \frac{2}{n_{\text{in}}}\right)$$

### PyTorch实现
```python
# ▶ 权重初始化
import torch
import torch.nn as nn

W = torch.empty(512, 256)

# Xavier 初始化
nn.init.xavier_uniform_(W)
print(f"Xavier: mean={W.mean():.4f}, std={W.std():.4f}")

# Kaiming 初始化 (for ReLU)
W2 = torch.empty(512, 256)
nn.init.kaiming_uniform_(W2, mode='fan_in', nonlinearity='relu')
print(f"Kaiming: mean={W2.mean():.4f}, std={W2.std():.4f}")
```

## 优化器的选择

### 默认推荐：AdamW
```python
# ▶ 优化器配置示例
import torch.optim as optim

# AdamW (推荐)
optimizer = optim.AdamW(
    model.parameters(),
    lr=1e-3,
    weight_decay=0.01
)
print(f"Optimizer: {optimizer.param_groups[0].keys()}")
```

### 大模型预训练：AdamW + 余弦调度
```python
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3, weight_decay=0.1)
scheduler = torch.optim.lr_scheduler.CosineAnnealingLR(optimizer, T_max=num_steps)
```

### 追求泛化：SGD + Momentum
```python
optimizer = torch.optim.SGD(
    model.parameters(),
    lr=0.1,
    momentum=0.9,
    weight_decay=1e-4
)
scheduler = torch.optim.lr_scheduler.MultiStepLR(
    optimizer, milestones=[30, 60, 90], gamma=0.1
)
```

## 学习率调优

### 学习率范围测试
1. 从很小学习率（如1e-7）开始
2. 指数增长到很大（如1e+1）
3. 画出loss曲线
4. 选择loss开始上升前的学习率作为上界

### 设置建议
| 模型大小 | 推荐学习率 |
|----------|-----------|
| 小型CNN | 1e-2 ~ 1e-3 |
| Transformer | 1e-4 ~ 1e-3 |
| 大模型(LLaMA) | 1e-5 ~ 3e-4 |
| ChatGLM微调 | 1e-6 ~ 1e-5 |

## 完整训练循环

```python
# ▶ 训练循环示例
import torch
import torch.nn as nn
import torch.nn.functional as F

model = nn.Linear(10, 2)
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3)

x = torch.randn(4, 10)
y = torch.randint(0, 2, (4,))

model.train()
optimizer.zero_grad()
logits = model(x)
loss = F.cross_entropy(logits, y)
loss.backward()
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
optimizer.step()

print(f"Loss: {loss.item():.4f}")
```
