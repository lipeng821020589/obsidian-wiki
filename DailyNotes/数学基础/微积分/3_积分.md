# 积分 (Integration)

## 基本概念

### 不定积分
求原函数：
$$\int f(x) dx = F(x) + C, \quad F'(x) = f(x)$$

### 定积分
$$\int_a^b f(x) dx = F(b) - F(a)$$

---

## 基本积分公式

| 函数 | 积分 |
|------|------|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$ ($n \neq -1$) |
| $\frac{1}{x}$ | $\ln|x| + C$ |
| $e^x$ | $e^x + C$ |
| $a^x$ | $\frac{a^x}{\ln a} + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\frac{1}{\cos^2 x}$ | $\tan x + C$ |

---

## 积分技巧

### 换元积分法
$$\int f(g(x)) g'(x) dx = \int f(u) du, \quad u = g(x)$$

### 分部积分法
$$\int u dv = uv - \int v du$$

---

## 多重积分

### 二重积分
$$\iint_R f(x, y) dA = \int_{a}^{b} \int_{c}^{d} f(x, y) dy dx$$

### 几何意义
- $\iint 1 dA = \text{面积}$
- $\iint f(x,y) dA$ = 曲面下体积

---

## 概率中的积分（期望）

### 连续型期望
$$\mathbb{E}[X] = \int_{-\infty}^{\infty} x f(x) dx$$

### 连续型方差
$$\text{Var}(X) = \int (x - \mu)^2 f(x) dx$$

---

## 深度学习中的积分应用

### 1. 概率分布归一化
$$\int_{-\infty}^{\infty} p(x) dx = 1$$

神经网络模型需要保证输出是概率分布。

### 2. 高斯分布的归一化因子
$$1 = \int \mathcal{N}(x; \mu, \sigma^2) dx = \int \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} dx$$

### 3. KL散度的积分形式
$$D_{KL}(P \| Q) = \int p(x) \log \frac{p(x)}{q(x)} dx$$

### 4. 变分推断
后验分布的边缘似然：
$$P(\text{data}) = \int P(\text{data} \mid \theta) P(\theta) d\theta$$

## 蒙特卡洛积分

### 原理
$$\int_a^b f(x) dx \approx \frac{b-a}{N} \sum_{i=1}^N f(x_i), \quad x_i \sim U(a,b)$$

### 更高效的方式（重要性采样）
$$\int f(x) dx = \int \frac{f(x)}{q(x)} q(x) dx \approx \frac{1}{N} \sum_{i=1}^N \frac{f(x_i)}{q(x_i)}$$

### 深度学习应用：ELBO
$$\mathcal{L} = \mathbb{E}_{q(z|x)}[\log P(x|z)] \approx \frac{1}{N} \sum_{i=1}^N \log P(x|z_i), \quad z_i \sim q(z|x)$$
