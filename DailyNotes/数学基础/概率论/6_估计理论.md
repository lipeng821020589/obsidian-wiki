# 估计理论

## 统计估计的基本问题

给定数据集 $\mathcal{D} = \{x_1, x_2, ..., x_n\}$，估计未知参数 $\theta$。

---

## 点估计 (Point Estimation)

### 定义
点估计是单个数值 $\hat{\theta}$ 作为 $\theta$ 的估计。

### 估计量的性质

| 性质 | 定义 |
|------|------|
| 无偏性 | $\mathbb{E}[\hat{\theta}] = \theta$ |
| 有效性 | $\text{Var}(\hat{\theta}_1) < \text{Var}(\hat{\theta}_2)$，更有效 |
| 一致性 | $n \to \infty$ 时，$\hat{\theta} \to \theta$ |
| 均方误差 | $\text{MSE}(\hat{\theta}) = \mathbb{E}[(\hat{\theta} - \theta)^2]$ |

### MSE分解
$$\text{MSE} = \text{Var}(\hat{\theta}) + \text{Bias}^2(\hat{\theta})$$

---

## 极大似然估计 (MLE)

### 似然函数
$$\mathcal{L}(\theta; \mathcal{D}) = P(\mathcal{D} \mid \theta) = \prod_{i=1}^n P(x_i \mid \theta)$$

### 对数似然
$$\ell(\theta; \mathcal{D}) = \log \mathcal{L}(\theta; \mathcal{D}) = \sum_{i=1}^n \log P(x_i \mid \theta)$$

### MLE定义
$$\hat{\theta}_{\text{MLE}} = \arg\max_\theta \mathcal{L}(\theta; \mathcal{D}) = \arg\max_\theta \ell(\theta; \mathcal{D})$$

等价于：
$$\hat{\theta}_{\text{MLE}} = \arg\min_\theta -\ell(\theta; \mathcal{D})$$

### 优点
- **一致性**：样本量大时收敛到真实参数
- **渐近正态**：$\sqrt{n}(\hat{\theta} - \theta) \to \mathcal{N}(0, I^{-1}(\theta))$
- **渐近有效**：在所有一致估计量中渐近方差最小
- **不变性**：若 $\hat{\theta}$ 是 $\theta$ 的MLE，则 $g(\hat{\theta})$ 是 $g(\theta)$ 的MLE

### 深度学习中的应用
几乎所有深度学习模型的训练都是MLE或其变体：
- 分类：交叉熵损失 = 负对数似然
- 自编码器：重构损失
- 语言模型：NLL损失

## 极大后验估计 (MAP)

### 贝叶斯公式
$$P(\theta \mid \mathcal{D}) = \frac{P(\mathcal{D} \mid \theta) P(\theta)}{P(\mathcal{D})}$$

### MAP定义
$$\hat{\theta}_{\text{MAP}} = \arg\max_\theta P(\theta \mid \mathcal{D}) = \arg\max_\theta P(\mathcal{D} \mid \theta) P(\theta)$$

### 与MLE的关系
$$\hat{\theta}_{\text{MAP}} = \arg\max_\theta \left[\log P(\mathcal{D} \mid \theta) + \log P(\theta)\right]$$

MAP = MLE + 对数先验 $\log P(\theta)$

### MAP的优势
- 引入先验知识
- 正则化效果（某些先验对应正则项）

### 例子：L2正则化 = 高斯先验的MAP

**假设**：权重 $\mathbf{w}$ 服从高斯先验 $\mathbf{w} \sim \mathcal{N}(0, \lambda^{-1}\mathbf{I})$

**先验对数**：
$$\log P(\mathbf{w}) \propto -\frac{\lambda}{2}\|\mathbf{w}\|_2^2$$

**MAP目标**：
$$\hat{\mathbf{w}}_{\text{MAP}} = \arg\max_\mathbf{w} \log P(\mathcal{D} \mid \mathbf{w}) - \frac{\lambda}{2}\|\mathbf{w}\|_2^2$$

这正是L2正则化的目标！

## 贝叶斯推断 (Bayesian Inference)

### 完全贝叶斯方法
不只估计一个点，而是求**后验分布**：
$$P(\theta \mid \mathcal{D}) = \frac{P(\mathcal{D} \mid \theta) P(\theta)}{P(\mathcal{D})}$$

### 后验预测分布
对新数据的预测：
$$P(x' \mid \mathcal{D}) = \int P(x' \mid \theta) P(\theta \mid \mathcal{D}) d\theta$$

### 近似贝叶斯推断
精确贝叶斯后验往往难以计算，需要近似方法：
- **变分推断 (VI)**：用简单分布近似后验
- **MCMC**：马尔可夫链蒙特卡洛采样
- **Laplace近似**：用高斯近似后验

## 变分推断 (Variational Inference)

### ELBO (Evidence Lower Bound)
$$\log P(\mathcal{D}) \geq \mathcal{L}(q) = \mathbb{E}_q[\log P(\mathcal{D} \mid \theta)] - D_{KL}(q(\theta) \| P(\theta))$$

### 优化目标
最大化 $\mathcal{L}(q)$ 等价于：
- 最小化 $D_{KL}(q(\theta) \| P(\theta \mid \mathcal{D}))$
- 使变分分布 $q(\theta)$ 逼近真实后验

### 应用：变分自编码器 (VAE)
```python
# VAE中的ELBO
def vae_loss(recon_x, x, mu, log_var, prior_mean=0, prior_var=1):
    # 重构损失（似然项）
    recon_loss = F.binary_cross_entropy(recon_x, x, reduction='sum')
    
    # KL损失
    kl_loss = -0.5 * torch.sum(
        1 + log_var - (mu - prior_mean)**2 / prior_var - log_var.exp()
    )
    
    # ELBO = 重构损失 - KL
    return recon_loss + kl_loss
```

## 经验风险最小化 (ERM)

### 定义
给定假设空间 $\mathcal{H}$，ERM选择：
$$\hat{h} = \arg\min_{h \in \mathcal{H}} \hat{R}(h) = \arg\min_{h \in \mathcal{H}} \frac{1}{n}\sum_{i=1}^n L(h(x_i), y_i)$$

### 训练误差 vs 泛化误差
- **训练误差**：$\hat{R}(h)$
- **泛化误差**：$R(h) = \mathbb{E}_{(x,y)}[L(h(x), y)]$
- **差距**：由VC维、复杂度等理论刻画

### 深度学习中的观察
深度学习往往**不**满足ERM经典理论（大参数量、仍然泛化），这催生了：
- 隐式正则化
- Lottery Ticket Hypothesis
- Double Descent

## 最大似然估计的深度学习实现

### 分类问题
```python
# MLE: 最大化 P(y|x)
# 等价于最小化负对数似然

logits = model(x)  # shape: (batch, num_classes)
# 负对数似然 = 交叉熵
loss = F.cross_entropy(logits, y)  # y是类别索引，不是one-hot
```

### 回归问题
```python
# ▶ 回归 MLE 损失
y_true = torch.tensor([1.5, 2.3, 0.8])
mu = torch.tensor([1.4, 2.5, 0.9])  # 模型预测

# 假设 sigma=1，负对数似然 = 0.5 * (y - mu)^2
loss = 0.5 * ((y_true - mu) ** 2).mean()
print(f"MSE Loss: {loss.item():.4f}")
```
