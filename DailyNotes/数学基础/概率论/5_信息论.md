# 信息论

## 自信息 (Self-Information)

### 定义
事件 $x$ 的自信息：
$$I(x) = -\log P(x)$$

### 理解
- $P(x)$ 越小，$I(x)$ 越大（稀有事件信息量大）
- $P(x) = 1$ 时，$I(x) = 0$（确定事件没有信息）
- $I(x) \geq 0$

### 对数底数选择
| 底数 | 单位 | 应用场景 |
|------|------|----------|
| 2 | bits | 信息论、编码 |
| $e$ | nats | 数学、深度学习常用 |
| 10 | digits | 工程 |

### 转换
$$\text{nats} = \text{bits} \times \log_2 e \approx \text{bits} \times 1.44$$

## 熵 (Entropy)

### 定义
随机变量 $X$ 的熵是其自信息的期望：
$$H(X) = \mathbb{E}[I(X)] = -\mathbb{E}[\log P(X)] = -\sum_x P(x) \log P(x)$$

### 离散情况
$$H(X) = -\sum_{i=1}^n P(x_i) \log P(x_i)$$

### 连续情况（微分熵）
$$H(X) = -\int f(x) \log f(x) dx$$

### 熵的性质
- $H(X) \geq 0$
- 均匀分布时熵最大
- $H(X) \leq \log |X|$（最大熵）

### 熵的几何意义
熵表示编码信息所需的最少比特数（对于离散变量）。

---

## 联合熵 (Joint Entropy)

### 定义
$$H(X, Y) = -\sum_x \sum_y P(x, y) \log P(x, y)$$

### 链式法则
$$H(X, Y) = H(X) + H(Y \mid X) = H(Y) + H(X \mid Y)$$

---

## 条件熵 (Conditional Entropy)

### 定义
给定 $Y$ 后 $X$ 的条件熵：
$$H(X \mid Y) = \mathbb{E}_Y[H(X \mid Y=y)] = -\sum_y P(y) \sum_x P(x \mid y) \log P(x \mid y)$$

### 简化形式
$$H(X \mid Y) = -\sum_{x,y} P(x, y) \log P(x \mid y)$$

### 链式法则
$$H(X_1, ..., X_n) = \sum_{i=1}^n H(X_i \mid X_1, ..., X_{i-1})$$

---

## 互信息 (Mutual Information)

### 定义
$$I(X; Y) = H(X) - H(X \mid Y) = H(Y) - H(Y \mid X)$$

### 对称形式
$$I(X; Y) = \sum_{x,y} P(x, y) \log \frac{P(x, y)}{P(x)P(y)}$$

### 另一种写法
$$I(X; Y) = \mathbb{E}_{P(x,y)}\left[\log \frac{P(x,y)}{P(x)P(y)}\right]$$

### Venn图表示
```
    [      H(X)      ]
    [    /      \     ]
    [ H(X|Y)  I(X;Y)  ]
    [    \      /     ]
    [   H(Y|X)         ]
    [       \ |       ]
    [     H(Y)        ]
```

### 性质
- $I(X; Y) \geq 0$
- $I(X; Y) = I(Y; X)$
- $I(X; Y) = 0$ 当且仅当 $X$ 和 $Y$ 独立

## 交叉熵 (Cross-Entropy)

### 定义
$$H(P, Q) = -\mathbb{E}_P[\log Q(X)] = -\sum_x P(x) \log Q(x)$$

### 与熵的关系
$$H(P, Q) = H(P) + D_{KL}(P \| Q)$$

### 深度学习中的应用
分类任务的损失函数：

```python
# PyTorch
loss = F.cross_entropy(logits, targets)

# 等价于：
probs = F.softmax(logits, dim=-1)
loss = -torch.mean(torch.sum(targets * torch.log(probs), dim=-1))
```

## KL散度 (Kullback-Leibler Divergence)

### 定义
$$D_{KL}(P \| Q) = \sum_x P(x) \log \frac{P(x)}{Q(x)} = \mathbb{E}_P\left[\log \frac{P(X)}{Q(X)}\right]$$

### 性质
- **非负**：$D_{KL}(P \| Q) \geq 0$
- **非对称**：$D_{KL}(P \| Q) \neq D_{KL}(Q \| P)$
- **不满足三角不等式**

### 与交叉熵的关系
$$D_{KL}(P \| Q) = H(P, Q) - H(P)$$

因此，最小化 $D_{KL}(P \| Q)$ 等价于最小化交叉熵 $H(P, Q)$（当 $H(P)$ 为常数时）。

### 深度学习应用：VAE的ELBO
$$\log P(x) \geq \mathcal{L} = -\mathbb{E}_{q(z|x)}[\log P(x|z)] + D_{KL}(q(z|x) \| P(z))$$

---

## JS散度 (Jensen-Shannon Divergence)

### 定义
$$JS(P \| Q) = \frac{1}{2} D_{KL}(P \| M) + \frac{1}{2} D_{KL}(Q \| M)$$

其中 $M = \frac{1}{2}(P + Q)$

### 性质
- $0 \leq JS(P \| Q) \leq 1$（对于以2为底的对数）
- **对称**：$JS(P \| Q) = JS(Q \| P)$

### 应用
- GAN（原始GAN使用JS散度）
- 文本生成评估（BLEU是另一种思路）

---

## 深度学习中的具体应用

### 1. 语言模型损失
$$\text{Loss} = -\frac{1}{T}\sum_{t=1}^T \log P(x_t \mid x_{<t})$$

这实际上就是负对数似然(NLL) = 交叉熵。

### 2. 标签平滑 (Label Smoothing)
软化标签，减少过拟合：
$$P_{\text{smooth}}(y) = (1 - \epsilon) P_{\text{hard}}(y) + \frac{\epsilon}{K}$$

等价于增加熵，降低置信度。

### 3. 知识蒸馏 (Knowledge Distillation)
学生网络学习教师网络的软标签：
$$\mathcal{L} = \alpha \cdot H(y_{\text{true}}, P_{\text{student}}) + (1-\alpha) \cdot H(P_{\text{teacher}}, P_{\text{student}})$$

其中 $H$ 是交叉熵。

### 4. InfoGAN
最大化潜在编码与生成数据之间的互信息：
$$\mathcal{L}_{\text{InfoGAN}} = \mathcal{L}_{\text{GAN}} - \lambda \cdot I(c; G(z,c))$$

### 5. Contrastive Loss
如SimCLR中使用的NT-Xent (Normalized Temperature-scaled Cross Entropy)：
$$\mathcal{L} = -\log \frac{\exp(\text{sim}(z_i, z_j)/\tau)}{\sum_{k \neq i} \exp(\text{sim}(z_i, z_k)/\tau)}$$
