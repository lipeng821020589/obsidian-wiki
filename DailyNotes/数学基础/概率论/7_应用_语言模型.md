# 语言模型中的概率论

## 语言模型的基本问题

给定词序列 $w_1, w_2, ..., w_T$，语言模型学习联合概率：

$$P(w_1, w_2, ..., w_T) = \prod_{t=1}^T P(w_t \mid w_1, ..., w_{t-1})$$

### 自回归分解
$$P(w_1, ..., w_T) = P(w_1) \cdot P(w_2 \mid w_1) \cdot ... \cdot P(w_T \mid w_1, ..., w_{T-1})$$

---

## Softmax与概率归一化

### 问题
语言模型输出的是 **logits** $z_t$，需要转为概率分布 $P(w_t \mid w_{<t})$。

### Softmax函数
$$P(w_t = i \mid w_{<t}) = \frac{\exp(z_t^i)}{\sum_{j=1}^{|V|} \exp(z_t^j)}$$

其中 $|V|$ 是词表大小（通常数万到数十万）。

### 数值稳定性
直接计算大词表的softmax可能溢出：

```python
def stable_softmax(logits):
    # 减去最大值避免溢出
    logits_max = logits.max(dim=-1, keepdim=True).values
    exp_logits = torch.exp(logits - logits_max)
    return exp_logits / exp_logits.sum(dim=-1, keepdim=True)
```

---

## 交叉熵损失 (Cross-Entropy Loss)

### 语言模型的训练目标
$$\mathcal{L} = -\sum_{t=1}^T \log P(w_t \mid w_{<t}) = -\sum_{t=1}^T \log \frac{\exp(z_t^{w_t})}{\sum_j \exp(z_t^j)}$$

### 等价形式
$$\mathcal{L} = -\sum_{t=1}^T \left(z_t^{w_t} - \log \sum_j \exp(z_t^j)\right)$$

### PyTorch实现
```python
# 方式1：直接用cross_entropy（推荐）
loss = F.cross_entropy(logits.view(-1, vocab_size), targets.view(-1))

# 方式2：手动计算NLL
probs = F.softmax(logits, dim=-1)
nll = -torch.log(probs.gather(-1, targets.unsqueeze(-1))).mean()
```

## 负对数似然 (Negative Log-Likelihood, NLL)

### 定义
$$\text{NLL} = -\log P(w_1, ..., w_T) = -\sum_{t=1}^T \log P(w_t \mid w_{<t})$$

### Perplexity (困惑度)
$$\text{PPL} = \exp\left(\frac{1}{T}\sum_{t=1}^T -\log P(w_t \mid w_{<t})\right) = \exp(\text{NLL/T})$$

**物理意义**：模型预测的不确定性（类似平均分支因子）

- PPL = 10：平均每步有10个等可能的选择
- PPL = 1：完美预测

```python
def calculate_ppl(loss):
    return torch.exp(loss).item()
```

## Label Smoothing (标签平滑)

### 硬标签的问题
- 正确类别概率为1，其他为0
- 导致模型过于自信

### 平滑方法
$$P_{\text{smooth}}(y \mid x) = (1 - \epsilon) \cdot \mathbb{1}[y] + \frac{\epsilon}{K}$$

其中 $K$ 是类别数，$\epsilon$ 是平滑因子（通常0.1）。

### 代码实现
```python
class LabelSmoothingCrossEntropy(nn.Module):
    def __init__(self, smoothing=0.1):
        super().__init__()
        self.smoothing = smoothing
    
    def forward(self, x, target):
        K = x.shape[-1]
        log_probs = F.log_softmax(x, dim=-1)
        
        # 平滑目标分布
        target_one_hot = torch.zeros_like(log_probs)
        target_one_hot.fill_(self.smoothing / (K - 1))
        target_one_hot.scatter_(1, target.unsqueeze(1), 1 - self.smoothing)
        
        loss = (-target_one_hot * log_probs).sum(dim=-1).mean()
        return loss
```

### 等价实现（更高效）
```python
# 使用nn.CrossEntropyLoss的label_smoothing参数
loss = nn.CrossEntropyLoss(label_smoothing=0.1)
```

## 采样策略

### Greedy Decoding
选择概率最高的词：
$$w_t = \arg\max_i P(w_t = i \mid w_{<t})$$

### Temperature Sampling
引入温度 $T$ 控制随机性：
$$P_T(w_t = i) = \frac{\exp(z_t^i / T)}{\sum_j \exp(z_t^j / T)}$$

| $T$ | 效果 |
|-----|------|
| $T < 1$ | 更确定，尖锐分布 |
| $T = 1$ | 原始分布 |
| $T > 1$ | 更随机，平滑分布 |

### Top-K Sampling
只从概率最高的$k$个词中采样：
```python
top_k_probs, top_k_indices = torch.topk(probs, k)
# 重新归一化
top_k_probs = top_k_probs / top_k_probs.sum()
# 采样
sampled_idx = torch.multinomial(top_k_probs, 1)
```

### Nucleus Sampling (Top-P)
从累积概率达到$p$的最小词集中采样：
```python
sorted_probs, sorted_indices = torch.sort(probs, descending=True)
cum_probs = torch.cumsum(sorted_probs, dim=-1)

# 找到达到p的边界
p_mask = cum_probs <= p
# 至少保留一个词
p_mask[..., -1] = True

# 重新归一化并采样
filtered_probs = sorted_probs * p_mask.float()
filtered_probs = filtered_probs / filtered_probs.sum()
sampled_idx = torch.multinomial(filtered_probs, 1)
```

## 束搜索 (Beam Search)

### 贪婪搜索的问题
只考虑当前最优，可能错过整体最优序列。

### 束搜索
维护$B$个最可能的候选序列（束）。

```python
def beam_search(model, start_token, beam_width=5, max_len=100):
    # 初始化
    sequences = [(start_token, 0.0)]  # (token_ids, log_prob)
    
    for _ in range(max_len):
        candidates = []
        for seq, score in sequences:
            # 获取下一个词的概率分布
            logits = model(seq)
            log_probs = F.log_softmax(logits[-1], dim=-1)
            
            # 扩展所有候选
            for i, log_p in enumerate(log_probs):
                new_score = score + log_p.item()
                candidates.append((seq + [i], new_score))
        
        # 选择top-B
        sequences = sorted(candidates, key=lambda x: x[1], reverse=True)[:beam_width]
    
    return sequences[0][0]  # 返回最优序列
```

### Length Penalty
避免倾向于短序列：
$$\text{score} = \frac{\log P(w_1,...,w_T)}{T^\alpha}$$

## KL散度在LM分析中的应用

### 模型蒸馏
教师-学生KL散度：
$$\mathcal{L} = (1-\alpha) H(y, P_{\text{student}}) + \alpha T^2 D_{KL}(P_{\text{teacher}/T} \| P_{\text{student}/T})$$

### 困惑度与熵的关系
$$H(P) = \frac{\text{NLL}}{T} = \log(\text{PPL})$$

## 代码：完整语言模型训练

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class LanguageModel(nn.Module):
    def __init__(self, vocab_size, d_model, n_heads, n_layers):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.transformer = nn.TransformerDecoder(
            nn.TransformerDecoderLayer(d_model, n_heads, batch_first=True),
            num_layers=n_layers
        )
        self.fc = nn.Linear(d_model, vocab_size)
    
    def forward(self, x, target=None):
        # x: (batch, seq_len)
        x = self.embedding(x)  # (batch, seq_len, d_model)
        x = self.transformer(x, x)  # 自注意力
        logits = self.fc(x)  # (batch, seq_len, vocab_size)
        
        loss = None
        if target is not None:
            # Cross-Entropy Loss = NLL
            loss = F.cross_entropy(
                logits.view(-1, logits.size(-1)),
                target.view(-1)
            )
        
        return logits, loss

# 训练
model = LanguageModel(vocab_size=50000, d_model=512, n_heads=8, n_layers=6)
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-4)

for epoch in range(num_epochs):
    for batch in dataloader:
        input_ids, labels = batch
        _, loss = model(input_ids, labels)
        
        optimizer.zero_grad()
        loss.backward()
        torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
        optimizer.step()
```
