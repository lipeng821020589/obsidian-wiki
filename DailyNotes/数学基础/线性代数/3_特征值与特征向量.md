# 特征值与特征向量

## 定义

对于方阵 $\mathbf{A} \in \mathbb{R}^{n \times n}$，如果存在标量 $\lambda$ 和非零向量 $\mathbf{v}$ 使得：

$$\mathbf{A}\mathbf{v} = \lambda\mathbf{v}$$

则：
- $\lambda$ 是 **特征值** (eigenvalue)
- $\mathbf{v}$ 是 **特征向量** (eigenvector)

### 几何意义
特征向量表示在矩阵变换后**方向不变**（只缩放）的向量。

---

## 特征方程

$$\mathbf{A}\mathbf{v} = \lambda\mathbf{v} \Rightarrow (\mathbf{A} - \lambda\mathbf{I})\mathbf{v} = 0$$

有非零解的条件：
$$\det(\mathbf{A} - \lambda\mathbf{I}) = 0$$

这个方程称为**特征多项式**，展开后是关于 $\lambda$ 的 $n$ 次多项式。

### 例子：2x2矩阵
$$\det\begin{pmatrix} a - \lambda & b \\ c & d - \lambda \end{pmatrix} = (a-\lambda)(d-\lambda) - bc = 0$$

---

## 特征值性质

1. **$n \times n$ 矩阵有 $n$ 个特征值**（可能有重复）
2. **特征值之和 = 迹**：$\sum_i \lambda_i = \text{tr}(\mathbf{A})$
3. **特征值之积 = 行列式**：$\prod_i \lambda_i = \det(\mathbf{A})$
4. **对称矩阵的特征值是实数**
5. **正定矩阵的特征值全为正**

---

## 特征向量性质

1. 特征向量之间**线性无关**（对应不同特征值）
2. 对称矩阵的特征向量可以正交
3. 归一化特征向量：$\|\mathbf{v}\|_2 = 1$

---

## 谱分解 (Spectral Decomposition)

对于**实对称矩阵** $\mathbf{A}$：

$$\mathbf{A} = \mathbf{Q}\mathbf{\Lambda}\mathbf{Q}^T$$

其中：
- $\mathbf{Q}$：正交矩阵，列是特征向量
- $\mathbf{\Lambda} = \text{diag}(\lambda_1, ..., \lambda_n)$：特征值对角矩阵

### 证明
因为 $\mathbf{Q}^T = \mathbf{Q}^{-1}$：
$$\mathbf{A}\mathbf{Q} = \mathbf{Q}\mathbf{\Lambda} \Rightarrow \mathbf{A} = \mathbf{Q}\mathbf{\Lambda}\mathbf{Q}^T$$

---

## 幂迭代 (Power Iteration)

求最大特征值及其特征向量的迭代算法。

### 算法
```
初始化: v = random_vector
重复:
    v_new = A @ v
    v_new = v_new / ||v_new||  # 归一化
直到收敛
```

### 收敛性
- 收敛到对应**最大特征值**的特征向量
- 收敛速度由 $\left|\frac{\lambda_2}{\lambda_1}\right|$ 决定

### 应用
- **PageRank**: Google的网页排名算法
- **PCA**: 求主成分
- **谱聚类**: 求特征向量

---

## 深度学习中的应用

### 1. PCA（主成分分析）
PCA的解是协方差矩阵的**最大$k$个特征值**对应的特征向量。

```python
import numpy as np

# 数据矩阵 X (n_samples, n_features)
# 求协方差
cov = X.T @ X / n_samples

# 特征值分解
eigenvalues, eigenvectors = np.linalg.eigh(cov)

# 取前k个主成分
top_k = eigenvectors[:, -k:]
X_pca = X @ top_k
```

### 2. 谱归一化 (Spectral Normalization)
控制神经网络权重矩阵的**谱范数**（最大奇异值）：

$$\mathbf{W}_{SN} = \frac{\mathbf{W}}{\sigma(\mathbf{W})}$$

其中 $\sigma(\mathbf{W})$ 是 $\mathbf{W}$ 的最大奇异值（等于 $\sqrt{\lambda_{\max}(\mathbf{W}^T\mathbf{W})}$）。

### 3. 扩散模型的谱分析
分数网络（score-based models）的谱性质影响生成质量。

### 4. 神经网络稳定性
- 权重矩阵特征值影响梯度传播
- 特征值大于1：梯度可能爆炸
- 特征值小于1：梯度可能消失
- **Spectral Norm** 用于正则化

## 代码实现

```python
import numpy as np
import torch

# NumPy
eigenvalues, eigenvectors = np.linalg.eig(A)  # 一般矩阵
eigenvalues, eigenvectors = np.linalg.eigh(A)  # 对称矩阵（更稳定）

# PyTorch
eigenvalues, eigenvectors = torch.linalg.eig(A)
eigenvalues, eigenvectors = torch.linalg.eigh(A)

# 求最大特征值（幂迭代）
def power_iteration(A, num_iter=100):
    b = torch.randn(A.shape[1], 1)
    for _ in range(num_iter):
        b_new = A @ b
        b_new = b_new / torch.norm(b_new)
        b = b_new
    return b

# 谱范数（最大奇异值）
sigma_max = torch.linalg.svd(A).S[0]
```
