## 🛠️ 使用方法

### 1. 在 Jupyter 中运行
下载 notebook 到本地，在 Jupyter 中打开运行：
```bash
# 克隆仓库
git clone https://github.com/udlbook/udlbook.git
cd udlbook/Notebooks
jupyter notebook
```

### 2. Google Colab
点击 GitHub 上的 **"Open in Colab"** 按钮直接在浏览器中运行。

### 3. Wiki 笔记
本目录下的精读笔记提供了：
- 公式推导详解
- 代码示例（含运行结果）
- 与其他教材的交叉引用
- 习题思路分析
